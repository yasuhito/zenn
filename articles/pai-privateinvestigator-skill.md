## PrivateInvestigator Skill とは

[PAI](https://zenn.dev/yasuhito/articles/pai-personal-ai-infrastructure) の人探しパック。**疎遠になった人との再接続** を目的とした、倫理的な人物調査を行う。

[OSINT Skill](https://zenn.dev/yasuhito/articles/pai-osint-skill) と似ているが、目的が異なる：

| スキル | 目的 |
|--------|------|
| OSINT Skill | ビジネス調査（採用、投資） |
| PrivateInvestigator | 個人的な再接続（友人、家族） |

## ユースケース

### 1. 昔の友人を探す

```
User: "高校の友人の田中太郎を探したい。最後に会ったのは2010年、東京在住だった"
```

### 2. 疎遠になった家族

```
User: "10年前に連絡が途絶えた従兄弟を探したい"
```

### 3. 恩師を見つける

```
User: "大学時代のゼミの先生、山田教授の現在を知りたい"
```

## 倫理フレームワーク

### 目的の確認

調査前に必ず目的を確認：

```
AI: "どのような目的で探していますか？"
    1. 久しぶりに連絡を取りたい
    2. お礼を言いたい
    3. 重要な知らせを伝えたい
```

### Red Lines（絶対禁止）

- ストーキング目的
- 嫌がらせ目的
- 法的紛争での利用
- 本人が連絡を拒否している場合

### 公開情報のみ

収集するのは **公開情報のみ**：

✅ 使用する情報源：
- LinkedIn、Facebook（公開プロフィール）
- 会社の公開ページ
- 論文、記事、講演記録
- 同窓会サイト

❌ 使用しない：
- 非公開の個人情報
- データブローカー
- ハッキングされたデータ

## ワークフロー

### 1. 情報収集

提供された情報を整理：

```
既知情報:
- 名前: 田中太郎
- 最終連絡: 2010年
- 当時の場所: 東京
- 当時の職業: エンジニア
```

### 2. 検索戦略

複数の角度から検索：

- フルネーム検索
- 名前 + 職業
- 名前 + 地域
- 名前 + 学校/会社

### 3. 候補者リスト

見つかった候補を信頼度付きでリスト：

| 候補 | 一致度 | 根拠 |
|------|--------|------|
| 田中太郎（LinkedIn） | 高 | 職歴・学歴一致 |
| 田中太郎（Facebook） | 中 | 年齢・地域一致 |
| T. Tanaka（論文） | 低 | 分野一致のみ |

### 4. 連絡方法の提案

直接の連絡先は提供しない。代わりに：

- 「LinkedIn で接続リクエストを送る」
- 「共通の知人経由で連絡する」
- 「公開されているビジネスメールに連絡する」

## プライバシー保護

### 最小限の情報収集

目的達成に必要な最小限の情報のみ収集。

### 情報の保持

調査完了後、収集した情報は削除を推奨。

### 相手の意思尊重

「見つけたが、連絡を望んでいない可能性がある」場合は警告。

## 出力例

```markdown
## 調査レポート: 田中太郎

### 候補者（信頼度: 高）

**LinkedIn プロフィール**
- 現在: 株式会社ABC シニアエンジニア
- 所在地: 神奈川県
- 経歴: 2010年頃に東京でエンジニアとして勤務（一致）

### 推奨連絡方法

1. LinkedIn で接続リクエスト（メッセージ付き）
2. 共通の知人: 佐藤花子さん（同じ会社の同僚）

### 注意事項
- 直接の連絡先（電話番号、住所）は取得していません
- 相手が連絡を望まない可能性を考慮してください
```

## まとめ

PrivateInvestigator Skill は PAI の人探し：

- **再接続目的** に特化
- **倫理フレームワーク** で悪用を防止
- **公開情報のみ** を使用
- **プライバシー保護** を重視

昔の大切な人とつながり直すために。
